Tập huấn GitHub

Chào mừng bạn đến với trang nội dung tập huấn về GitHub!

GitHub là một dịch vụ lưu trữ mã nguồn và quản lý phiên bản phân tán, được sử dụng rộng rãi trong cộng đồng phát triển phần mềm. Tập huấn này sẽ giúp bạn hiểu rõ hơn về cách sử dụng GitHub để quản lý mã nguồn, hợp tác với đồng đội, và tham gia vào các dự án mã nguồn mở.

Trong tập huấn này, bạn sẽ học được các khái niệm cơ bản về Git, cách tạo ra repository, thực hiện commit, tạo nhánh, và thực hiện các thao tác hợp nhất (merge). Bạn cũng sẽ tìm hiểu về các tính năng phổ biến của GitHub như pull requests, issues, và wiki.
